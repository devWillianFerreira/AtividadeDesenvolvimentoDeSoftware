Sobre o projeto

O objetivo deste trabalho é desenvolver um portfólio básico utilizando as linguagens HTML e CSS. O projeto tem como finalidade apresentar de forma simples e organizada informações como nome, descrição pessoal, habilidades. A proposta visa exercitar os conhecimentos iniciais em estruturação de páginas web com HTML e aplicação de estilos com CSS, criando uma interface visual agradável.
